函数调用
JS（ES5）里面有三种函数调用形式：

  • func(p1, p2)
  • obj.child.method(p1, p2)
  • func.call(context, p1, p2) // 先不讲 apply

一般，都知道前两种形式，而且认为前两种形式「优于」第三种形式。
但第三种调用形式，才是正常调用形式：
func.call(context, p1, p2)
