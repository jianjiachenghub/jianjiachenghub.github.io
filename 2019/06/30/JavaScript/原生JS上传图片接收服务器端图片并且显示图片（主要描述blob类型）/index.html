1.了解后端处理图像的方式

一：图片以独立文件的形式存储在服务器的指定文件夹中，再将路径存入数据库字段中
二：将图片转换成blob，直接存储到数据库的 Image 类型字段中（这种方式负担很大不建议使用）

blob (binary large object)二进制大对象
