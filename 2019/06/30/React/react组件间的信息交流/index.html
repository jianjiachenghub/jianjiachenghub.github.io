React 组件之间交流的方式，可以分为以下 3 种：
a【父组件】向【子组件】传值；
b【子组件】向【父组件】传值；
c 没有任何嵌套关系的组件之间传值（PS：比如：兄弟组件之间传值）

1.父组件向子组件传递信息
